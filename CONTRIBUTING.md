# Guia de Contribuição

Para manter a integridade do Antigravity Skills Ultimate, siga estas regras:

1. **Forks apenas**: Não peça acesso de escrita direto.
2. **Conventional Commits**: Use padrões como `feat:`, `fix:`, `docs:`.
3. **Branchs**: Crie branchs descritivas (ex: `feature/nova-skill`).
4. **PRs**: Todo Pull Request passará por revisão manual antes do merge.
